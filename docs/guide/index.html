# Objetivo

El objetivo de este documento es mostrar la instalación y modo de uso de BTServicesExposer, el cual es una nueva capa de integración que brinda seguridad y características adicionales en el consumo del API Bantotal y de todos los servicios que se publican mediante BTServices Publisher. Esta capa implementada en Node.js, genera y publica en base a los WSDLs de los servicios, una API RESTful y su correspondiente especificación en Swagger. Permite ser instalada en servidores independientes brindando mayor seguridad a la infraestructura.
